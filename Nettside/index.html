Project goal

Provided by Kongsberg Discovery, the goal of the project is to deliver an automated system for applying two-component adhesives to cermic sonar components.

Hardware

ViscoTec

We are using a ViscoTec pree-flow eco-DUO 330 as our glue dispenser.
Using this hardware, we will be able to doese extremely small amounts of glue.
This is important as the SY50 is very small. The smallest dosing quantity is 0.005ml.
