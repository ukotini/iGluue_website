Project Introduction

How it is done today:

Kongsberg Discovery are manually manually apllying the glue to the sonar,
SY50. The SY50 is small, only 11-13 mm in diameter and 3-5 mm in hight.
There is 150% more glue applied to the SY50, this reduces the sonars
ability to function as intended.

Our solution:

We will develop an automated system for applaying two-component adhesives
to cermiac sonar components using ViscoTech.

The iGluue team

Our hardware

ViscoTech

ViscoTech is a two-component adhesive system.
It is used to apply glue to the sonar.
The system is controlled by a PLC.
The PLC is programmed in Codesys.
The system is connected to a computer.
The computer is programmed in C#.
